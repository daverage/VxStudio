================================================================================
                    WHERE TO PUT PLUGIN FILES
================================================================================

IMMEDIATE: macOS Only

1. Build plugins:
   cd /Users/andrzejmarczewski/Documents/GitHub/VxStudio
   cmake --build build -j$(nproc)

2. Package for distribution:
   mkdir -p vxsuite-dist/vxsuite-1.0.0-macos-universal/{VST3,AudioUnits}
   cp -r build/*VST3/*.vst3 vxsuite-dist/vxsuite-1.0.0-macos-universal/VST3/
   cp -r build/*AU/*.component vxsuite-dist/vxsuite-1.0.0-macos-universal/AudioUnits/
   # Add license, install instructions, etc.
   cd vxsuite-dist
   zip -r vxsuite-1.0.0-macos-universal.zip vxsuite-1.0.0-macos-universal/

3. Copy to website:
   cp vxsuite-1.0.0-macos-universal.zip web/downloads/
   sha256sum vxsuite-1.0.0-macos-universal.zip > web/downloads/checksums.txt

4. Update HTML:
   Edit web/downloads/index.html:
   - Update version number (if different)
   - Update file size
   - Update SHA256 checksum
   - Update release notes

================================================================================
                        DEPLOYMENT OPTIONS
================================================================================

OPTION 1: Self-Hosted (Recommended)
  ✅ Full control
  ✅ High performance
  ❌ Server management required
  
  Setup:
    rsync -avz --delete web/ user@vxstudio.marczewski.me.uk:/var/www/vxstudio/
  
  See: WEB_DEPLOYMENT_GUIDE.md (Option 1)

OPTION 2: Cloudflare Pages (Easiest)
  ✅ Free SSL/HTTPS
  ✅ Global CDN
  ✅ Auto-deploys on git push
  ❌ Requires GitHub
  
  Setup:
    1. Push web/ to GitHub
    2. Connect Cloudflare Pages
    3. No build command needed
    4. Done!
  
  See: WEB_DEPLOYMENT_GUIDE.md (Option 2)

OPTION 3: Netlify (Also Easy)
  ✅ Similar to Cloudflare
  ✅ Good free tier
  ❌ Requires GitHub
  
  Setup:
    1. Push web/ to GitHub
    2. Connect Netlify
    3. No build command needed
    4. Done!
  
  See: WEB_DEPLOYMENT_GUIDE.md (Option 3)

HTTPS: REQUIRED
  Use Let's Encrypt (free, automatic renewal)

LARGE FILES: Don't commit ZIPs to GitHub
  Instead:
    - Upload directly via rsync/SFTP, OR
    - Use GitHub Releases for binary storage, OR
    - Use external CDN (Cloudflare Pages handles this)
