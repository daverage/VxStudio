📋 Installation Instructions

macOS

  1. Download the .zip file
  2. Extract the archive
  3. Copy .vst3 bundles to /Library/Audio/Plug-Ins/VST3
  4. Copy .component files to /Library/Audio/Plug-Ins/Components
  5. Restart your DAW
  6. Rescan plugins in your DAW settings

Windows (When Available)

  1. Download the .zip file
  2. Extract to a temporary folder
  3. Run the installer executable
  4. Choose VST3 plugin directory
  5. Restart your DAW
  6. Rescan plugins in your DAW settings
